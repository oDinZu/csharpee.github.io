Present Symphony

Now, the direction of my intentions have evolved since 2013 and the technology or tools working in parallel with enable more control building unique experiences for each idea or project.

In short summary of this method, only critical pieces of code are added relative to the project needs and wants.

The core pieces of each applications focus is:

  • Content First
  • Accessibility
  • Security
  • Performance
  • Adaptability
Nowandays, web app developers are calling this type of method 'JAMstack'.

“The Jamstack is not about specific technologies. It's a new way of building websites and apps that delivers better performance, higher security, lower cost of scaling, and a better developer experience.”—JAMstack

The JAMstack is an application stack that stands for Jekyll, API's & Markup. Some JAMstackers replace this with complex Javascript pancakes like React or VueJs.

I have moved away from hefty frameworks; many of these systems are bulky and each one has its own steep learning curve for the user, developer, designer and administrator. I discovered it is more efficient & friendlier to piece together ones own system or software stack.

Easing the content creators experience sharing information through online information portals is my focus. There is genius & elegance in simplicity!

“Truth is ever to be found in simplicity, and not in the multiplicity and confusion of things.” —Isaac Newton
